## 已知bug：
 - 通话时期点击屏幕最左上角挂断通话。通话时期需通过左上角的挂断按钮挂断，但因bug，按钮无法显示，但依旧能触发。
 - 忘记密码功能无法使用
 - 进入通话时按Back键后将无法结束通话，只能强制结束进程
 - 添加好友成功后无提示信息，需重启app
